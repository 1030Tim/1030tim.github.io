人工智慧

相信大家在這幾年頻繁的聽到人工智慧
（artificial intelligence）俗稱AI

  • AI掃把
  • AI電鍋

乎什麼東西都可以被灌上AI兩個字了
要想要暸解AI，我想先問問何為智慧

有意識或無意識的執行行為，
判斷、思考、預測等都是屬於智慧

人工智慧就是透過模仿人類的思考及智慧
而人類智慧就是由學習而來的
說專業一點人工智慧就是利用
機器學習實現的
而機器學習又分為三個類別

Supervised learning

監督式學習
回想我們小時候是如何學習的，舉例來說：
當我們在路上看到一輛擁有四個輪子、前方有兩個燈的交通工具時，
我們猜這是機車，媽媽告訴我們：「這是汽車。」（正確答案Label)
經過一次一次猜測，一次次驗證
，我們學會了辨識汽車的特徵，
並能夠區分汽車與其他交通工具。
這個過程與機器學習的監督式學習的方式類似，
我們可以將這些觀察到的資訊轉換為「特徵」（Features）：

有正確答案對照的都叫做監督式學習

Unsupervised Learning

無監督學習
沒有標籤，讓機器自行分析數據結構，發現數據之間的關聯性或分群（Clustering）。
例子：我們觀察交通工具的不同特徵，自己歸類哪些是汽車、機車或卡車
後續會慢慢介紹，白話點就是讓機器自學。

Reinforcement Learning

強化學習
透過獎勵與懲罰機制，讓機器學習最佳決策，常應用於遊戲AI、自動駕駛等領域。
例子：學騎腳踏車時，跌倒後知道哪種平衡方式才是正確的
一樣後面也會詳細說明，白話就是正確給你獎勵錯誤給你懲罰。

ML 學習筆記

機器學習（ML, Machine Learning）是找出資料中的複雜函式與規律。
機器學習主要有三個分類回歸、分類、結構化學習

Regression

當模型的輸出是一個連續數值時，這稱為回歸。例如：
房價預測（輸入：房屋面積、房齡；輸出：價格）
銷售量預測（輸入：天氣、廣告投放量；輸出：銷售數量）

Classification

分類是讓機器在有限選項中做出選擇。例如：
電子郵件分類（輸入：郵件內容；輸出：垃圾郵件 / 非垃圾郵件）
圖片識別（輸入：圖片；輸出：貓 / 狗）

Structured Learning

除了數值預測與分類，機器學習還可以產生有結構的輸出，例如：
生成圖像（如 DALL·E）
自動寫作（如 ChatGPT）
自動翻譯（如 Google 翻譯）

機器學習的核心步驟

找出函式與模型設計

找到合適的函式需要具備一定的領域知識（Domain Knowledge）。

Regression Model

Simple Linear Regression

定義函式：
線性回歸模型：
$$
y = wX + b
$$
其中：
(y) 是我們要預測的數值（目標變數）
(X) 是輸入變數
(w) 為權重（weight），(b) 為偏差（bias）
定義 Loss（損失函數）：
Loss 是衡量預測誤差的函數，例如：
均方誤差（MSE, Mean Squared Error）：
$$
MSE = \frac{1}{n} \sum_{i=1}^{n} ( y_i - \hat{y_i} )^2
$$
平均絕對誤差（MAE, Mean Absolute Error）：
$$
MAE = \frac{1}{n} \sum_{i=1}^{n} | y_i - \hat{y_i} |
$$

Optimization

目標是讓 Loss 變得最小，主要方法是梯度下降（Gradient Descent）：
而梯度下降是屬於黑箱優化演算法
初始化權重$$（隨機選擇一組 ( w_0, b_0 )）$$
計算損失函數的梯度（求導數得到斜率）
根據梯度更新參數
若梯度為負數，則增加 ( w )
若梯度為正數，則減小 ( w )
缺點：可能陷入局部最小值（Local Minima）

進階最佳化方法

學習率（Learning Rate）調整：
若學習率太大，可能錯過最佳解（oscillation）。
若學習率太小，收斂速度變慢。
改進方法：
Momentum（動量法）
Adam Optimizer（適應性學習率調整）
